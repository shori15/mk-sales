# MK売上管理

東京MKハイヤー・タクシー用の売上管理PWA。

## 機能
- タブ売上 / 日報売上 / 空車高速(自腹)を1日3項目で記録
- 21日締め(21日〜翌20日)サイクルでの累計集計
- 月間目標と達成率
- CSVエタスポート / インポート
- iPhoneのホーム画面に追加してネイティブアプリのように使える
- データは各端末のローカルに保存(プライバシー重視)

## 使い方
1. iPhoneのSafariで以下のURLを開く: `https://shori15.github.io/mk-sales/`
2. 共有ボタン → 「ホーム画面に追加」
3. ホーム画面のアイコンをタップして起動

## データの扱い
- データは各iPhone内のlocalStorageに保存されます
- サーバーには一切送信されません
- 機種変更時はCSVエクスポート → 新端末でCSVインポート
