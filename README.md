# MK ハイヤー料金計算

東京MKハイヤー・タクシー用の料金計算PWA。

## 機能

- **羽田定額** — エリア別定額料金・超過料金計算
- **成田定額** — 送迎タイプ別定額料金・超過料金計算
- **ハイヤー貸切** — 予約時刻・実乗車時刻入力、時間制vs距離制の自動比較
- **契約** — エミレーツ航空・ユナイテッド航空（UAクルー）の契約料金・未収現収判定
- **ルールブック** — 料金ルール・条件をカテゴリ別に検索

## 料金表準拠

2026年5月21日〜

## 使い方

iPhoneのホーム画面に追加してネイティブアプリのように使える（PWA対応）。

## 対応車種

- ラグジュアリークラス（ベンツV / グランエース / アルファード）
- エグゼクティブクラス（BMW7 / レクサスLM500 / ベンツS550・S400 / ハイエース）
- ファーストクラス（ロールスロイス / マイバッハGLS600）
